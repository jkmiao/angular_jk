#### 1. 安装 angular-cli
官网指引: https://github.com/angular/angular-cli

` npm install -g angular-cli`
` ng --help`

#### 2. 项目初始化
- 'ng new habo_v2'
- 'cd habo_v2'
- 'ng serve'

- 打开浏览器: http://localhost:4200

#### 3. git 版本管理
- `git init`
- `mkdir docs`
- `vi quick-start.md`
- `git add docs`
- `git commit -s -m 'add first version'`
